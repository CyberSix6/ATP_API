Introduction

This document describes how to interface with your Teledyne LeCroy quantumdata instrument using our Python API.

The API can be used on any host that runs either Python 2 or 3. This includes Linux, Windows and MacOS. You may also use the API directly on the quantumdata instrument - provide 'localhost' for the IP address.

Installation

Step 1: Download our API

Link

Step 2: Install Python (2 or 3) on your system.

Step 3: Install Paramiko

Paramiko is optional, but necessary if you plan on using ssh for secure connections. Use this link for installation instructions.

HDMI Source Tests

The API allows a quantumdata instrument user to run our HDMI Source tests without the GUI. While our GUI was designed to step a user through each test, the API was designed to control every test and iteration so that the programmer decides which tests and iterations to run, in the order of their chosing.

There is no need to provide a CDF file when using the API with HDMI source tests. Many tests allow the programmer to specify attributes, such as a VIC or bit-depth.

DP Source Tests

The API allows a quantumdata instrument user to run our DP Source tests without the GUI. While our GUI was designed to step a user through each test, the API was designed to run one full test and if it requires input during the test it uses callback for user input to proceed further in the test.

Programmer should provide a CDF file when using the API with DP source tests.
